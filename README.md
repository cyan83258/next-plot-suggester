# 다음 전개 추천 (Next Plot Suggester)

SillyTavern 채팅 기록을 바탕으로 다음 전개를 추천해주는 확장 프로그램입니다.

## 버전
- v1.2.0

## 주요 기능

### 1. 다음 전개 추천
- 현재까지의 채팅 내용을 분석하여 다음에 어떤 전개가 가능할지 추천합니다
- 추천 내용은 확장 프로그램의 프롬프트만 사용하여 생성됩니다 (기존 시스템 프롬프트 제외)
- 추천 개수와 각 추천의 문장 수를 직접 숫자로 설정할 수 있습니다

### 2. 버튼 형태 UI
- 추천 내용이 파스텔 하늘색/흰색 테마의 버튼으로 표시됩니다
- 버튼 클릭 시 해당 내용이 클립보드에 자동 복사됩니다
- 복사 후 추천 메시지가 자동으로 사라집니다

### 3. 장르/스타일 설정
- 코미디, 일상, 느와르, 판타지, 로맨스 등 다양한 장르 선택 가능
- 사용자 정의 장르 추가 기능 (영문/한글 이름 지정)
- 선택한 장르가 추천 생성에 반영됩니다

### 4. API 설정
- SillyTavern의 현재 연결된 API 사용 또는 커스텀 API 엔드포인트 지정 가능
- Connection Manager의 프로필 선택 기능 (프로필별 API 사용 가능)
- API 연결 테스트 기능 제공

### 5. 자동 추천 (선택)
- 새 메시지 수신 시 자동으로 추천 생성 (설정에서 on/off)

## 사용 방법

### 설정 열기
1. 확장 메뉴(퍼즐 아이콘) 클릭
2. "다음 전개 추천 설정" 선택

### 추천 생성
1. 확장 메뉴에서 "다음 전개 추천" 클릭
2. 또는 채팅 입력창 옆의 전구 아이콘 클릭

### 추천 사용
1. 표시된 버튼 중 원하는 추천 클릭
2. 클립보드에 자동 복사됨
3. 입력창에 붙여넣기하여 사용

## 설정 옵션

| 설정 | 설명 |
|------|------|
| 기능 활성화 | 확장 프로그램 전체 on/off |
| 새 메시지 시 자동 추천 | AI 응답 후 자동으로 추천 생성 |
| 추천 당 문장 수 | 각 추천의 길이 (1-10) |
| 추천 개수 | 생성할 추천 수 (1-10) |
| 장르/스타일 | 추천 생성 시 반영할 장르 |
| 추가 지시사항 | 프롬프트에 추가할 커스텀 지시 |
| 연결 프로필 | 사용할 Connection Manager 프로필 |
| API 연결 방식 | 현재 API 또는 커스텀 API |
