선택지 자동 생성 확장

SillyTavern Extension for Next Plot Suggestions

소개

채팅의 맥락을 분석하여 다음 전개에 대한 선택지를 자동으로 생성해주는 SillyTavern 확장 프로그램입니다. 캐릭터 설정, 월드 인포, 채팅 기록 등을 종합하여 상황에 맞는 창의적인 선택지를 제안합니다.

주요 기능

  • 다음 전개 추천 채팅 컨텍스트를 분석하여 자연스러운 다음 전개 선택지를 생성합니다.
  • 다양한 입력 소스 지원 캐릭터 설명, 페르소나, 월드 인포, 시나리오 요약, AU-World-Builder 등을 조합하여 더 정확한 추천을 제공합니다.
  • 장르 기반 추천 원하는 장르를 선택하면 해당 장르에 맞는 전개를 추천합니다.
  • 커스텀 프롬프트 추가 지시사항을 입력하여 원하는 방향으로 추천을 유도할 수 있습니다.
  • 유연한 설정 문장 수, 추천 개수 등을 자유롭게 조절할 수 있습니다.

지원 장르

12가지 기본 장르를 제공하며, 커스텀 장르도 추가할 수 있습니다.

코미디 일상 느와르 판타지 로맨스 호러 미스터리 액션 드라마 SF 스릴러 모험

사용 방법

1 SillyTavern에 확장 프로그램을 설치합니다.
2 채팅 화면 우측의 확장 패널에서 "다음 전개 추천" 버튼을 클릭합니다.
3 설정에서 사용할 LLM Provider와 모델, 장르 등을 선택합니다.
4 생성된 선택지 중 마음에 드는 것을 클릭하면 입력창에 자동으로 붙여넣기 됩니다.
